### Dependencies Installation

#### Windows

1. Install [Scoop](https://scoop.sh/).
2. Install [Java (Corretto)](https://scoop.sh/#/apps?q=corretto-jdk).
3. Install [Maven](https://scoop.sh/#/apps?q=maven).
